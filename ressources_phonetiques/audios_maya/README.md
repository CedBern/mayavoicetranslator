# README — Corpus audio maya (DOReCo Yucatec)

Ce dossier contient :
- Les fichiers audio du corpus DOReCo Yucatec (format .wav)
- Les métadonnées associées (CSV, TXT)
- Un script Python pour indexer, exploiter et préparer les données pour l’API, l’UI ou la recherche

## Structure
- `doreco_yuca1254_audiofiles_v2.0/` : audios et métadonnées
- `index_corpus_maya.py` : script d’indexation et d’exploitation

## Utilisation rapide
1. Lancer le script Python pour générer un index JSON des audios et métadonnées
2. Utiliser l’index pour l’API, l’UI, la transcription ou l’analyse linguistique

## Ressources complémentaires
- Phonologie et grammaire : voir `proto_mayan_phonology.json`
- Orthographe ALMG : voir section dédiée dans le JSON
