# Archives des serveurs API MayaVoiceTranslator

Ce dossier contient les anciennes versions et variantes du serveur API, conservées pour historique, transparence et référence technique.

## Fichiers archivés

- **api-server-production.js** : version de production antérieure, remplacée par `api-server.js`.
- **api-server-enhanced.js** : version expérimentale avec fonctionnalités avancées, fusionnées dans la version principale.
- **api-server-simple.js** : version simplifiée pour tests ou déploiements légers.
- **api-server-simple-backup.js** : sauvegarde d’une version minimale, non maintenue.

## Référence actuelle

La version de référence et maintenue de l’API est désormais :

- **api-server.js** (racine du projet)

Toutes les évolutions, corrections et déploiements doivent se faire à partir de ce fichier.

## Pourquoi archiver ?

- Garder une trace de l’évolution technique du projet
- Permettre la consultation ou la restauration d’anciennes logiques si besoin
- Éviter la confusion pour les nouveaux contributeurs
